STED Optimization documentation
===============================

This software can be used for online optimization of STED imaging parameters.
The current implementation supports Abberior STED microscopes through the Imspector Python Interface `SpecPy <http://imspectordocs.readthedocs.io/en/latest/specpy.html>`_.
